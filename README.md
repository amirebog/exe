## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/amirebog/exe.git
```

Navigate to the project:

```bash
cd exe
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

---

## 🚀 Deployment

Deploy instantly with Vercel:

```bash
npm run build
```
